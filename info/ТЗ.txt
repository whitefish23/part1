Необходимо выполнить верстку согласно макета используя методы и решения на усмотрение исполнителя.
Разрешается использовать плагины и любые инструменты разработки.

- поле "мое имя фамилия" текст должен быть изменен на данные исполнителя.
- в блоке "Стек знаний" по умолчанию должны быть "зачеканы" те навыки, которыми обладает исполнитель, при необходимости 
можно добавить дополнительные поля навыков
- В поле "о себе" нужно кратко рассказать о себе заменив исходный текст на свой. Фотографию заменить по желанию сохраняя стилизацию
- Блок "Заметки о задании" должен иметь следующий функционал:
	1) при заполнении поля ввода и нажатия кнопки "добавить заметку" должна добавляться новая заметка в блок ниже
	2) после перезагрузки страницы заметки должны оставаться (подразумевается реализация функционала на клиенте, без использования бд)
	3) при переполнении поля для заметок должен появляться стилизованный скроллбар
	4) должна быть реализована возможность редактировать текст внутри блока заметки, отредактированный текст так же должен оставаться после перезагрузки страницы.
	5) при нажатии на крестик заметки должны удалятся
- Блочные новости должны быть ссылками, при клике на кнопку «подробнее» должны выводиться alert с текстом «clicked» без срабатывания ссылки.
